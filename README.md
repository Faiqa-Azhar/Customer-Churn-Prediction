# 📊 Customer Churn Prediction using Machine Learning


## 📌 Project Overview

Customer churn is one of the biggest challenges faced by companies. Losing customers directly impacts revenue, and businesses need early prediction systems to identify customers who are likely to leave.

This project builds an **AI-powered Customer Churn Prediction System** that predicts whether a customer is likely to churn based on customer behavior, service usage, and financial information.

The project includes:

- Machine Learning model for churn prediction
- Customer segmentation
- Churn probability prediction
- Business recommendations
- Interactive Streamlit dashboard



# 🎯 Problem Statement

Companies often lose customers without knowing the reasons behind their decisions.

The goal of this project is to answer:

**"Which customers are likely to leave, and what actions can businesses take to retain them?"**

## Data Preprocessing

Steps performed:

- Handling missing values
- Encoding categorical features
- Feature transformation
- Scaling numerical features


## Model Training

Models experimented:

- Logistic Regression
- Random Forest
- XGBoost


Final selected model:

**XGBoost Classifier**


Reason:

- Good performance on structured data
- Handles complex relationships
- Works well for business prediction problems
